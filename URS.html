ЧИСЛА
NUM-001.01 Типы чисел Должны поддерживаться как целочисленные так и дробные числа. О
NUM-001.02 Для дробных чисел разделителем является символ точки ".". О
NUM-002.01 Знак числа Должны поддерживаться как положительные, так и отрицательные числа. О

ИНТЕРФЕЙС. КЛАВИШИ.
INT-K-001 Клавиши ввода числа Ввод числа производится посредством нажатия на экранные клавиши от "0" до "9". О
INT-K-002 Дополнительные клавиши ввода 1)Для ввода дробной части используется клавиша ".".
2)Для смены знака числа используется клавиша "+/-". При нажатии на клавишу происходит смена знака числа на противоположный.		 О
INT-K-003 Клавиши выбора арифметических операций 1) Для выполнения операций используются клавиши:
"+" - для суммирования,
"-" - для вычитания,
"*" - для умножения,
"/" - для деления
2) Кроме того, если до нажатия клавиши выбора арифметической операции уже были введены два числа A и B, то необходимо выполнить предыдущую арифметическую операцию и ее результат будет операндом A. Т.е. при последовательном вводе 1 + 2 + 4, вначале будут введены числа A= 1, B=2, при нажатии следующего "+" будет произведено суммирование A и B, результат помещен в A=3 и следующая операция будет 3 + 4. (более подробно смотри раздел "Выполнение арифметических операций")		 О
INT-K-004 Клавиша сброса Клавиша "С" предназначена для очистки полученных результатов и ошибок, и сброса состояния калькулятора на первоначальное. О
INT-K-005 Клавиша BackSpace Клавиша "<<" используется для удаления последнего значения числа отображаемого на дисплее. При этом удаление символа применяется именно к отображаемому числу, вне зависимости от того, как оно было получено: введено с клавиатуры или получено в результате арифметических действий. О
INT-K-006 Клавиша выполнения вычисления Клавиша "=" предназначена для выполнения:
- вычисления значения введенных двух чисел
- автоматического вычисления последней операции с числом
(более подробно смотри раздел "Выполнение арифметических операций")		 О
ИНТЕРФЕЙС. ДИСПЛЕЙ ВЫВОДА.
INT-D-001 Отображение результата Отображаемая длина числа не должна превышать 18 символов без учета знака минуса. О
ВЫПОЛНЕНИЕ АРИФМЕТИЧЕСКИХ ОПЕРАЦИЙ
AR-001 Суммирование Программа должна выполнять сложение двух введенных с клавиатуры чисел: A + B, а также суммировать последнее полученное в результате арифметических действий число с числом введенным с клавиатуры. О
AR-002 Вычитание Программа должна выполнять вычитание двух введенных с клавиатуры чисел: A - B, а также вычитать из последнего полученного в результате арифметических действий числа число введенное с клавиатуры. О
AR-003 Умножение Программа должна выполнять умножение двух введенных с клавиатуры чисел: A * B, а также умножать последнее полученное в результате арифметических действий число с числом введенным с клавиатуры. О
AR-004 Деление Программа должна выполнять деление двух введенных с клавиатуры чисел: A / B, а также делить последнее полученное в результате арифметических действий число на число введенное с клавиатуры. О
AR-005 Начало вычисления Процесс вычисления должен начинаться либо по нажатию клавиши "=" либо по нажатию одной из клавиш арифметических операций. При этом условием начала вычислений должно является наличие двух чисел, введенных ранее, или наличие результата предыдущей арифметической операции на дисплее и введенного числа с клавиатуры. О
AR-006 Повторное вычисление с константой В случае, если необходимо произвести вычисление, где операнд B является константой, а операнд A является полученным значением, должна использоваться клавиша "=". При этом результат помещается в операнд A, а операнд B и арифметическая операция остаются неизменными. Каждое нажатие клавиши "=" производит автоматическое повторение последней операции с константой B. Например: 2+2=<4> = <6> = <8> ... О
AR-007 Проверка ошибок Должны обрабатываться две ошибки:
1) Деление на ноль. В этом случае на экране должно отобразится сообщение "Деление на ноль"
2) Переполнение. Если целая часть числа превышает 18 знакомест без учета знака, то должно отобразится "Ошибка переполнения"
В случае любой из ошибок ввод любых чисел или выполнение операций запрещено.
Выход из состояния ошибки должен производится только по нажатию на клавишу сброса "С".		 О
